🐙 GitHub Deployment Guide
Host MediVault free on GitHub Pages
1

Create GitHub Account

Go to github.com and sign up free.

2

Create Repository

Click + → New repository → name it medivault → Public → Add README → Create.

3

Upload This File

Click Add file → Upload files → rename to index.html → Commit to main.

4

Enable GitHub Pages

Settings → Pages → Source: Deploy from branch → main → /(root) → Save.

5

Live App

After 2 minutes: https://YOUR-USERNAME.github.io/medivault/
